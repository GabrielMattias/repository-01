Os 3 típos de estilização:


CSS "In-line";


CSS "Interno";


CSS "Externo".



Qual é a diferença?


CSS "In-line": É quando uma tag HTML (como "p", por exemplo) é estilizada individualmente.


CSS "Interno": É quando todas as iterações de uma tag HTML são estilizadas usando uma única tag "style", colocada na parte "head" do arquivo HTML.


CSS "Externo": É quando as tags de um arquivo HTML são estilizadas a partir de um arquivo CSS externo.
